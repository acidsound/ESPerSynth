# Roland TR-808 드럼 머신 알고리즘 분석 연구 계획

## 연구 목표
Roland TR-808의 각 드럼 소스별 실제 알고리즘과 음색 특성을 상세히 분석하여 기술 보고서 작성

## 분석 대상 드럼 소스
1. 킥 (Kick)
2. 스네어 (Snare) 
3. 찰국 (Cymbal)
4. 틸프 (Tilt)
5. 오펜ated 킬드럼 (Opended Kick)
6. 오펜ated 스네어 (Opended Snare)
7. 리어스 (Rear)

## 각 드럼에 대해 분석할 항목
- 신디사이즈 메커니즘
- ADSR 엔벨롭 설정
- 소음 생성 방법
- 반사파 특성
